# HalqaManager

HalqaManager is an Android application developed with Kotlin that helps manage study circles effectively. It allows users to organize, plan, and interact with study groups in a user-friendly interface.

## Features
- **Circle Management**: Create, edit, and delete study circles.
- **Member Management**: Add and remove members from circles.
- **Event Scheduling**: Schedule events and meetings for study groups.
- **Notifications**: Get notified for upcoming meetings and group activities.
- **User-Friendly Interface**: Built with a focus on user experience, making navigation and functionality intuitive.

## Installation
To install HalqaManager, clone the repository and run the application in Android Studio:

```bash
git clone https://github.com/osamadijdjd/HalqaManager.git
```

## Usage
- Open the app and create a new study circle.
- Invite members by sharing the circle link.
- Schedule meetings and manage events easily.
